dotfiles
========

My dotfiles

## `n`

Use to initialize new Node.js project with `babel`, `standard`, `tape` supports.

### Usage

Copy `templates/n` directory to your `~/templates` before the first use. Run:

```sh
$ ./n your-project-name
$ cd your-project-name
```
Edit `package.json` and do your work :)
